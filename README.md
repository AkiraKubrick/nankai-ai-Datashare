# 期末资料整理分享
## 1、项目性质
南开大学人工智能本科生自主管理与更新的资料整理与分享活动，完全无偿，用于相关学习资料的跨年级分享
## 2、项目目的
打破复习资料信息差，让大多数同学能够轻松得到往年复习资料，花费更少的时间来完成无意义的重复劳动，从而将更多的时间用于对自身发展有利的活动中去。
## 3、获取资料前应认同
所有的资料都是学长学姐们无偿整理，并且经过了一定的审核和补充的，我们只将资料用于复习和交流，任何人不能进行买卖等商业行为，如果有任何意见和补充请联系我们。
## 4、加入我们
项目为动态更新和实时调整的，所以每年都需要新的成员加入，我们需要您有资料收集和整理能力、媒体宣传能力、项目管理能力（其中一种即可）\
我们的分工暂时有：资料整理者、宣传者、项目管理者（暂时由项目发起人以及大四同学担任），每一种分工只是按照各自擅长的技能划分，不区分贡献度。\
加入我们后会按照技能进行分工，然后开展各自的工作，最后将我们的工作无偿贡献出来供全部同学使用和学习，我们希望您也是一个具有开源精神的分享者。\
目前我们缺少：24级、25级进行资料整理的同学(整理大一和大二相关资料)，以及不限年级的宣传同学。
## 5、说明
再次强调我们的项目不收取任何费用，任何商业活动或者收费活动均不是我们项目成员承办，也不应该用以我们项目的名称。\
整个项目的开展过程中，不应涉及政治、暴力、反社会、黄色等违反道德和法律的信息，项目的管理者将会严格审核和管理。
