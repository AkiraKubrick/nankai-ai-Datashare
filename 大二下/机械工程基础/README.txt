这个课是整个南开独一份的机械类课程，全程几乎都在念ppt，教学质量一般，也是自动化相对比较头疼的专属课程。22级和23级是一起上的。

这个课主要由平时考勤加作业（20）、期中考试（30）、期末考试（50）组成。
首先是考勤。我们那年可能赶上教学评估了或者其他的，老师考勤次数比较多，同时在某些课结束都会有课堂考勤题，通常是简答题，有些题比较开放。
作业是每次都留，前几次可能会有solidworks画图，需要自学，不过题目是可以在网上找到的。

其次是期中考试。这个期中考试说是考试，但其实并不严格，老师管的不严，学生可能也不认真对待，座位也不岔开，并且是开卷。我记得22和23级是三道题，我记得其中两道分别计算齿轮的传动比、绘制机构运动简图并计算自由度，第三道是关于平面连杆机构的，要满足什么样的条件。每题10分，老师改卷子是相当严格的，我记得有些同学可能才得了十几分，这可是实打实扣在总分上的，所以最后的成绩可能不尽人意。

最后是期末考试，在考试前老师会有几节课复习，不过是跳着读ppt罢了……ppt有很多东西，大部分都记不住，看能记住什么，考试你觉得会考什么就记什么吧，考试的时候也至少不会答偏。我记得考试有问答，有计算。东西很杂，但是也意味着一些很细的东西没必要看，问答出了一个”液压传动的优势“之类的题目，大家可以参考着出题风格复习。计算题我记得就那些题型：自由度计算、平面连杆机构的组成条件、齿轮传动比计算、液压计算等。作业题会考。

大家一定一定要重视期中考试！！！
