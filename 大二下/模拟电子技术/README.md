## 模电资料整理
1、主要整理了ppt和往年题，以及实验部分的两个实验报告（大多数实验都是当场看效果），平时作业没有整理，因为用的是公版教材，所以可以在网上搜到标准答案   
2、25年考试中，题型为选择+解答题，选择题考察较为细致，其中不乏PN结相关知识点，复习时需要额外注意一下零碎知识点，占分较高。 解答题比较中规中矩，题型和考察知识点与参考题中差别不大，难度适中，结合课后题和参考题以及20种常见电路的一般分析方法复习，解答题不会很难拿分。
3、模电是两位老师轮流讲课，各讲一半，各有风格，讲课方式、快慢都不太一致，需要大家自己花时间学习。模电作为电子信息、控制相关专业的必修课，难度较难而且实践性较高，如果有想要深耕相关领域的同学，一定要打牢模电基础。
4、老师最后会留一节课给所有班级画一些解答题的重点（其实就是20种常见电路），请大家留意。
5、最后推荐一个网课，可以用于期末复习参考视频， b站网址：<https://b23.tv/qQqgmZz>，视频所讲内容较为简单，不涉及零碎知识点，但是对于解答题的讲解还是较为清晰（个人感受，仅供参考，当时拿这个视频期末速通了模电）
6、模电需要用到电路基础相关知识点，如果此部分知识不太熟练，可能会影响到课程进度，上课听不懂老师在讲什么，建议提前把电路相关内容再看一看。
