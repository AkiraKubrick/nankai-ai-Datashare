课件和教材：本课程的教材原版尚可一看，但是中文翻译版（也就是本课程的参考教材）翻译堪称灾难且课后习题找不到官方的中文答案，完全不推荐拿来阅读
课件也是灾难级别，完全不成体系+夹杂大量英文+各种模棱两可的表述，也不适合拿来自学。课件不包含考试完全不涉及的第八章内容（讲的是P-NP问题等）

作业题答案：包含英文原版书作者在个人网站上给出的英文版答案，以及笔者自己电子留档的第5-7章的答案

复习建议：由于去年直到最后一节课都仍然在讲课，没有留出时间押题或者复习，本课程没有任何复习资料
，本人一年前手写的总结资料也已经遗失，因此此处仅根据个人回忆在readme中给出复习建议

考试题目全部是大题，基本都是手写算法或者让你进行推导或证明，不仅会考察PPT中的部分原题以及推导和证明，
还会考察算法思想（比如出一些PPT上完全没出现过的题目，单纯死记硬背原题很难得分，必须搞懂某一类算法在干什么）

第一章 需要记住G-S匹配算法的步骤、其变种的步骤以及这个算法的部分结论
第二章 要掌握算法时间复杂度的排序，有些排序是反直觉的需要特别注意
第三章 直接死记硬背树上给出的各个案例即可，基本只能考这些
第四章到第七章 这些章节很容易出PPT上面完全没有的题目，单纯死记硬背题目的作用不大
建议先通过AI或者其他优秀的自学资源搞懂这些算法的底层逻辑，再去学PPT上面的案例效果更好，
特别是第四章贪心算法除了会分析还要会证明贪心最优
