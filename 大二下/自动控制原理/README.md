## 自动控制原理复习
1、自动控制原理是一门前后逻辑非常严谨的课程，在学习和复习时一定要把握整个课程的主要脉络。  
2、如果是方老师的班级，那就平时就尽量跟进方老师的讲课进度，方老师是整个本科讲课最好的老师，yyds！如果一直跟着方老师的进度，期末基本不用花太多时间去复习自控，知识点非常明确。  
3、自控的题型是填空+解答，填空分值特别重，需要特别关注，25年考试时就有同学后面解答全对但是填空错太多而分低。  
填空题主要考察一些细小知识点，需要把ppt多复习几遍。    
解答题复习时就很固定了，每一章哪些内容能够出大题考察其实很明确，自己学习的时候能感受出来，老师最后复习课也会再说，25年好像考察了根轨迹、离散系统稳定性判据、校正  
4、复习时主要参考ppt例题和作业题即可，解答题难度不大，与平时作业题难度差不多，认真复习搞懂每一步最后考试一般都能写出来。25年属于是题目难度不大，但是计算费劲，按计算器都要很长时间。
